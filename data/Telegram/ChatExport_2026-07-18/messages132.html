Для решения вашей задачи можно использовать метод многомерного шкалирования (MDS) для визуализации расстояний между брендами и дополнительную информацию о размерах точек. Вот как это можно сделать:

### Шаг 1: Подготовка данных
1. Создайте матрицу сопряженности (совместного выбора) для 50 брендов. Это будет матрица 50x50, где элемент \( M_{ij} \) обозначает количество раз, когда бренды \( i \) и \( j \) были выбраны вместе.
2. Создайте вектор частот выборов для каждого бренда. Это будет вектор длиной 50, где элемент \( F_i \) обозначает количество раз, когда бренд \( i \) был выбран.

### Шаг 2: Вычисление расстояний
3. На основе матрицы сопряженности вычислите расстояния между брендами. Один из способов сделать это — использовать обратное значение количества совместных выборов как расстояние (например, \( D_{ij} = 1 / (M_{ij} + 1) \)).

### Шаг 3: Многомерное шкалирование (MDS)
4. Примените метод MDS к матрице расстояний, чтобы получить координаты точек в 2D-пространстве. В Python это можно сделать с помощью библиотеки scikit-learn:

import numpy as np
from sklearn.manifold import MDS

# Примерные данные: замените на вашу матрицу сопряженности и вектор частот
M = np.array([[...], [...], ...])  # Матрица сопряженности 50x50
F = np.array([...])  # Вектор частот длиной 50

# Вычисление расстояний
D = 1 / (M + 1)

# Применение MDS
mds = MDS(n_components=2, dissimilarity='precomputed', random_state=42)
positions = mds.fit_transform(D)

# positions теперь содержит координаты точек в 2D-пространстве

### Шаг 4: Визуализация
5. Используйте matplotlib для создания графика, где координаты точек берутся из positions, а размеры точек определяются значениями из вектора частот.

import matplotlib.pyplot as plt

# Примерный код для визуализации
plt.figure(figsize=(10, 10))

# Нормализация размеров точек для визуализации
sizes = 1000 * (F / F.max())

# Создание графика
plt.scatter(positions[:, 0], positions[:, 1], s=sizes, alpha=0.5)

# Добавление меток к точкам
for i, (x, y) in enumerate(positions):
    plt.text(x, y, f'Brand {i+1}', fontsize=9)

plt.title('MDS Visualization of Brands')
plt.xlabel('MDS Dimension 1')
plt.ylabel('MDS Dimension 2')
plt.grid(True)
plt.show()

Этот код выполнит следующие действия:
- Построит координаты точек в 2D-пространстве с помощью MDS.
- Создаст график, где точки представляют бренды, размеры точек пропорциональны частоте выбора бренда, а расстояния между точками отражают частоту их совместного выбора.
