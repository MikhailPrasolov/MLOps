## Ограничения методологии

1. **Теги** — подсчёт основан на хештегах, которые авторы ставят вручную. Многие вакансии без тегов (76.9% для seniority) — статистика по ним недоступна.
2. **Word boundaries (v2)** — пересчитано с `\b` (см. раздел 6). FP contamination: VK −98%, Сбер −65%, ivi −79%, English −88%, BI −52%. **Не удалять substring v1 counts — нужны для сравнения.**
3. **Зарплаты** — парсим только 29.2% вакансий (другие не имеют границ или используют формат, который regex не понимает).
4. **Общие русские слова** (Финансы, Страхование, Воронка) — substring даёт 99-100% FP. Использовать только после word boundaries или заменять на BI / 1С / конкретный инструмент.
