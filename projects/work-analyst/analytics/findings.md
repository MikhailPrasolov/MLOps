## Ограничения методологии

1. **Теги** — подсчёт основан на хештегах, которые авторы ставят вручную. Многие вакансии без тегов (76.9% для seniority) — статистика по ним недоступна.
2. **Поиск компаний и скиллов** — substring search без word boundaries, есть false positives. Для точной статистики нужен NLP/ner.
3. **Зарплаты** — парсим только 29.2% вакансий (другие не имеют границ или используют формат, который regex не понимает).
4. **VK count (4968)** — завышен из-за substring "вк".
